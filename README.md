# Lotteria Kiosk (Python)

이 프로젝트는 롯데리아 키오스크의 기본 주문 과정을  
콘솔 환경에서 구현한 Python 프로그램입니다.  
메뉴 선택, 수량 입력, 총합 계산 등을 통해  
Python의 기초 문법과 흐름 제어를 연습하기 위해 제작했습니다.

---

## 주요 기능

- 메뉴 목록 출력  
- 메뉴 선택 및 수량 입력  
- 총 금액 계산  
- 주문 내역 요약 출력  
- 콘솔 기반의 간단한 사용자 인터페이스

---

##제작 목적

Python의 기초 문법과 기본 로직 구성을 연습하기 위한  
학습용 프로젝트입니다.  

다음 개념들을 익히는 데 목적이 있습니다.

- 입출력 처리(input)  
- 조건문  
- 반복문  

## 사용 기술

- Python 3  
- 기본 입출력 처리  
- 조건문, 반복문

## 참고 사항

이 프로젝트는 GUI나 데이터베이스 기능 없이  
기본 로직 구현에 집중한 학습용 프로젝트 입니다.
