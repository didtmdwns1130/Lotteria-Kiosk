이 프로젝트는 롯데리아 키오스크의 기본 주문 과정을
콘솔 환경에서 구현한 Python 프로그램입니다

메뉴 선택
수량 입력
총합 계산 등을 통해
Python의 기초 문법과 흐름 제어를 연습하기 위해 제작했습니다


1. 주요 기능
메뉴 목록 출력
메뉴 선택 입력
수량 입력
총 금액 계산
주문 내역 출력
콘솔 기반의 간단한 사용자 인터페이스

2. 제작 목적
Python 기초 문법과 기본 로직 구성을 연습하기 위한 학습용 프로젝트입니다
입출력 처리
조건문 반복문
활용 등을 이해하는 것을 목표로 하였습니다

3. 참고 사항
이 프로젝트는 학습용으로 제작되었으며
GUI나 데이터베이스 기능 없이
기본 로직 구현에 집중하였습니다
